## 🛠️ Configuración del Proyecto

### 🔧 Requisitos Previos

- Node.js
- npm
- Hardhat

### 📝 Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/fquinteroc/Proyecto-1-Hardhat.git
   cd Proyecto-1-Hardhat

2. Instala las dependencias
   ```bash
   yarn install

3. Ejecuta el proyecto
   ```bash
   yarn test
